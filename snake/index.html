Instructions

This is simple snake game and easy to play. The move the snake press Arrow up key to move snake upside, Arrowdown to move down, Arrowright to move right and Arroeleft to move left. To set the initial size snake enter the desired number in range of 2-5. If no length is provided or is out range the default size is provided. The game finishes if snake touches it's own body or collide with border. The snake length increases if it's eat the food. To pause the game click the mouse. To start the game click the mouse and snake starts moving downwards. To start the game click the start button below. There is also a score board to keep record of your game.

Thank You!!! Enjoy the game!!!
